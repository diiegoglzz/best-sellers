# Amazon best sellers analysis
Análisis de los libros mas vendidos en Amazon usando Python y Pandas

## Estructura del proyecto
best_sellers/

├── data/

│   └── bestsellers.csv

├── file_io.py       # carga de datos

├── logic.py         # análisis

├── charts.py        # visualizaciones

└── main.py          # punto de entrada

## Tecnologías
- Python 3
- Pandas
- Matplotlib
- Seaborn

## Como ejecutar
```bash
python3 -m venv venv
source venv/bin/activate
pip install pandas matplotlib seaborn
python3 main.py
```

## Análisis incluidos
- Top libros por rating
- Autores con más apariciones
- Distribución por género
- Evolución por año
